# Schema Core Library

## Getting Started
- Install the [dotnet 8 SDK](https://dotnet.microsoft.com/en-us/download/dotnet/8.0)
- Open `Schema.sln` in your IDE and build.
- Run tests:
  ```bash
  dotnet test Schema.Core.Tests/Schema.Core.Tests.csproj
  ```
